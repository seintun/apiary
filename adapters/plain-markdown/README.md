# Plain Markdown Adapter

Use this when there is no agent runtime.

1. A human or single assistant acts as coordinator.
2. Copy `templates/scout-brief.md` into separate chats or assign to humans.
3. Ask each scout to return `templates/scout-output.yaml` shape.
4. Fill `templates/synthesis-report.md`.
5. Verify manually.
6. Save the final decision in your docs/wiki/repo.
