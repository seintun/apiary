# Example: Tool Adoption Analysis

## Prompt

Should we adopt Tool X into our assistant workflow?

## Apiary setup

Scouts:
- research: what Tool X does and evidence
- devil's advocate: risks, overlap, instability
- adaptation: minimal native pattern if useful

## Expected synthesis

- adopt, reject, or borrow patterns only
- list risks and mitigations
- define next step
- write back only reusable decision/process
